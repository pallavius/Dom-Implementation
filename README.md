🎨 DOM Color Boxes & Greeting App

A simple interactive web application built using HTML, CSS, and JavaScript that demonstrates the use of DOM manipulation.
Users can enter their name to receive a greeting and interact with colored boxes that change their background color when clicked.

This project was created as part of my JavaScript learning assignment in my MERN Stack journey.

🚀 Project Overview

The objective of this project is to practice DOM manipulation and event handling using JavaScript.

The webpage includes:

A greeting header

Interactive color boxes

A user input field

A greeting button

Users can enter their name and receive a personalized greeting, and clicking on any box will change its background color dynamically.

✨ Features
🔹 Greeting Section

Displays "Hello" initially.

Users can enter their name in the input field.

Clicking the Greet button updates the header to:

Hello, [User Name]
🔹 Interactive Color Boxes

Four boxes are displayed with labels:

Red

Blue

Green

Yellow

When a user clicks on any box, the background color changes to the respective color.

Example:

Clicking Red box → background becomes red

Clicking Blue box → background becomes blue
